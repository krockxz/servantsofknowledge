About API Tokens

API tokens allow you to authenticate with the REPUB API programmatically. Use your token in the Authorization header:

Authorization: Token YOUR_TOKEN_HERE

Keep your token secure! Anyone with your token can access your account via the API.

API Usage Examples
Python (requests)
import requests

headers = {
    'Authorization': 'Token {{ token.key|default:"YOUR_TOKEN_HERE" }}'
}

response = requests.get('{{ request.build_absolute_uri }}/api/', headers=headers)
cURL
curl -H "Authorization: Token {{ token.key|default:"YOUR_TOKEN_HERE" }}" \
     {{ request.build_absolute_uri }}/api/
